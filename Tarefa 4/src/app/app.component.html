1. Crie 3 novos componentes: GameControl, Impar e Par
  2. O componente GameControl deve ter botões que iniciam e pausam o jogo
  3. Os componentes Impar e Par devem conter apenas um texto (par/ímpar) + um número
  4. Quando o jogo começa, um evento (que possui um número que é incrementado++) deve ser emitiddo a cada segundo
  5. O evento deve ser "listenable (escutável)" de fora do componente
  6. Quando o jogo é pausado, nenhum evento deve ser emitido (clearInterval(this.xxx))
  7. Para cada número par emitido um componente Par deve ser criado mostrando o texto + número, o mesmo para número ímpar
  8. Exemplo:
  9. Ímpar - 1
  10. Par: 2
  11. Estilize a cor do texto dos componentes Par e Ímpar

Dicas:

AppCompoente = componente pai

setInterval(() => { this.eventEmitter.emit(this.lastNumber + 1); this.lastNumber++; }, 1000)

Usar @Output() (quando quer emitir algo) e @Input() (quando algo vem de fora)
